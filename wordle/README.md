# Wordle by Santo ㊗️
 
## **Objective :**
  - To guess the 5 letter word 
## **RULES :**
  -  ⚝  You have 10 guesses
  -  ⚝  The probabilityChart array will show how close you are to the word
  -  ⚝  `2`  =  exact character position
  -  ⚝  `1`  =  character position is wrong
  -  ⚝  `0`  =  character is not part of the word
